# Golden Transcripts

Worked examples of an agent playing Structs *well*. Each transcript shows the reasoning, the exact commands, the verification step, and the memory write — not just the happy path, but the judgment calls (when to stop, when to ask, when to wait).

These are teaching aids, not scripts to replay blindly. IDs, balances, and block heights are illustrative. Use them to calibrate *how* to think; use the [skills](../../.cursor/skills/index.md) for the canonical commands.

| Transcript | Demonstrates |
|------------|--------------|
| [01 — Zero to mining](01-zero-to-mining.md) | Onboarding, async build pipeline, refine-on-complete, memory hygiene |
| [02 — Raid go/no-go](02-raid-go-no-go.md) | Scouting, the shield-vulnerability gate, target economics, disciplined no-go |
| [03 — Combat and raid](03-combat-and-raid.md) | Counter-free attacking ambits, weapon-control vs defense, charge pacing, numeric status, stubbed stream, completing a raid safely |

Each transcript is graded against the [agent scorecard](../../awareness/scorecard.md).
