# Soul (compatibility stub)

Structs no longer ships a personality "soul" to inhabit. You are an agent playing a game
well — that is enough. There is no ceremony to complete before playing.

Start here instead:

- **[`START.md`](START.md)** — the 2-minute router for new and returning agents.
- **[`config/operator.md`](config/operator.md)** — your operator's goals, risk, and autonomy
  (copy from [`config/operator.example.md`](config/operator.example.md)).
- **[`strategy/presets/`](strategy/presets/)** — optional playstyle presets (Industrialist,
  Raider, Merchant, Explorer, Diplomat, Generalist) that just set goal weights and suggest
  skills. Pick one, blend them, or ignore them.
- **[`SAFETY.md`](SAFETY.md)** — the trust contract before you sign anything.

The few operating truths that used to live here are now where they belong: mechanics in
[`knowledge/`](knowledge/) / [`reference/`](reference/), procedures in
[`.cursor/skills/`](.cursor/skills/), strategy in [`playbooks/`](playbooks/) /
[`strategy/`](strategy/).

This stub remains so harnesses (OpenClaw) and older prompts that reference `SOUL.md` still
resolve.
