# Skills

10 CLI-grounded procedures for playing Structs. Each skill uses `structsd tx structs` and `structsd query structs` commands.

| Skill | Purpose |
|-------|---------|
| [Onboarding](structs-onboarding/SKILL) | Address registration, planet exploration, first builds |
| [Mining](structs-mining/SKILL) | Ore extraction and refining |
| [Building](structs-building/SKILL) | Construction, activation, movement, defense, stealth |
| [Combat](structs-combat/SKILL) | Attacks, raids, defense setup |
| [Exploration](structs-exploration/SKILL) | Planet discovery, fleet movement |
| [Economy](structs-economy/SKILL) | Reactor staking, providers, agreements, allocations |
| [Guild](structs-guild/SKILL) | Guild creation, membership, Central Bank |
| [Power](structs-power/SKILL) | Substations, allocations, player connections |
| [Diplomacy](structs-diplomacy/SKILL) | Permissions, address management |
| [Reconnaissance](structs-reconnaissance/SKILL) | Intelligence gathering, query catalog |
